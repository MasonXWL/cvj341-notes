CSS Anatomy

CSS Blocks

This is a CSS block:

selector { property: value; }

  • The selector is the HTML element we want to style
  • This can be:
    • a specific element (<p>)
    • an id (#preamble)
    • a class (.branches-of-gov)
  • Enclosed within the curly braces ({}) is the declaration
  • We can specify multiple declarations for each selector
  • There are hundreds of properties we can change
  • Properties and values are separated by a colon (:)
  • All declarations should have a semicolon (;) at the end
